<h2>Description</h2>
The project is a very simplistic entry-level program featuring a few basic Lua programming concepts and functions. The project was developed to work with the cross-platform Lua framework, Solar2D.

<br />

<h2>Languages and Utilities Used</h2>

- <b>Lua</b> 

<h2>Environments Used</h2>

- <b>Solar2D</b> (cross-platform framework)

- <b>When playing on Mobile:</b> Tilt left/right to move the spaceship in that direction.
<br>
- <b>When playing throgh Solar2D:</b> Touch and drag the space ship to move in in the direction that you are dragging it towards.
